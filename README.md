# biblioteca
Trabalho de DW sobre API REST

Nesse trabalho, usamos Node, React e MySql.

Para funcionar, é necessário criar um banco de dados, no MySql, na sua máquina com o nome "biblioteca" (código no backend/banco/biblioteca.sql), alterar as informações user e
password no arquivo backend/util/db.js com as suas informações, abrir um cmd para frontend/ e um para backend/ e dar "npm install" e "npm run dev" em cada um (pota padrão do 
backend: 5000, porta padrão do fronend: 5173), além de manter ambos abertos para não parar de rodar. Abra o link de localhost que o cmd enviou, vá para  página registrar e 
coloque um email e uma senha de, pelo menos, seis dígitos. Faça login com o email e senha utilizados no registro. Vá para o perfil e cadastre um livro. Volte para a página 
inicial e clique no livro que criou para ler ele.
